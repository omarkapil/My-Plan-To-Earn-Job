```

   - chmod 755 myfile.txt (rwxr-xr-x)
```

```
3. Change ownership:
   sudo chown newuser:developers myfile.txt
```

**Hands-on Lab 3D: SSH Hardening (1.5 hours)**

**Secure SSH:**
```
1. Generate SSH key:
   ssh-keygen -t ed25519 -C "admin@server"

2. Copy to server:
   ssh-copy-id -i ~/.ssh/id_ed25519 user@host

3. Edit SSH config:
   sudo nano /etc/ssh/sshd_config
   
   Changes:
   PermitRootLogin no
   PasswordAuthentication no
   X11Forwarding no
   MaxAuthTries 3

4. Restart SSH:
   sudo systemctl restart ssh

5. Test:
   - Key auth: should work
   - Password: should fail
   - Root: should fail
```

**Shell Scripting (2 hours):**
